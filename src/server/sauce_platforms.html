sauce_platforms

Makes it easier to deal with the platforms offered by Sauce Labs testing systems.

Loosely based on browsers with reasonable usage: http://caniuse.com/usage_table.php

Sauce Labs current supported platforms:

  • https://saucelabs.com/platforms
  • (in JSON form) http://saucelabs.com/rest/v1/info/browsers/webdriver
